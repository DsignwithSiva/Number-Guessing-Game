# 🎮 Number Guessing Game (Java Swing GUI)

A simple and interactive **Number Guessing Game** built using **Java Swing**. The game challenges the player to guess a randomly generated number between 0 and 100.

---

## 🧠 How the Game Works

- The system randomly generates a number between **0 to 100**
- The player enters a guess via the GUI input box
- After each guess:
  - 🎯 If the guess is correct → A success dialog is shown with a victory message
  - 📉 If the guess is too low → A hint is shown
  - 📈 If the guess is too high → A hint is shown
- Player can:
  - 🔁 Start a **New Game**
  - ❌ Click **Give Up** to reveal the number and end the game

---

## 📊 Features

- 🎲 Random number generation from 0 to 100
- 💬 Hints: "Too Low" / "Too High"
- 🔁 New Game button to restart anytime
- 🚪 Give Up option to exit and reveal the correct number
- 📈 Statistics:
  - Number of attempts
  - Highest score
  - Time taken by the player
